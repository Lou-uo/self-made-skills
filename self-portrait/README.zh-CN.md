# 自我画像（Self-Portrait）技能

一个用于创建和维护用户自我画像档案的综合性技能，帮助实现个性化的AI助手体验。

## 概览

自我画像技能帮助用户构建详细的个人档案，记录以下信息：
- 技术专长和经验水平
- 工作风格和偏好设置
- 学习目标和职业发展规划
- 沟通协作模式
- 问题解决方法论

## 功能特性

### 🎯 档案管理
- **创建档案**：从零开始构建完整的用户画像
- **更新信息**：持续完善和扩展档案内容
- **导出查看**：生成可读性强的档案摘要
- **导入恢复**：从备份文件恢复现有档案

### 🤖 智能助手
- **个性化响应**：根据用户经验水平调整解释深度
- **自适应沟通**：匹配用户的沟通风格偏好
- **目标导向帮助**：协助内容与用户目标保持一致
- **模式识别**：从交互历史中学习用户行为模式

### 📊 分析洞察
- **技能差距分析**：识别需要提升的领域
- **学习建议**：推荐相关资源和学习路径
- **职业规划**：提供发展路径建议
- **进度追踪**：监控技能和职业发展进度

## 快速开始

### 1. 创建您的档案
```
我想创建一个自我画像档案。我是一名拥有3年React和Node.js经验的软件工程师。
```

### 2. 添加技能信息
```
在我的档案中添加Python、Docker和AWS。我正在学习GraphQL和TypeScript。
```

### 3. 定义工作风格
```
我偏好详细的解释和实际示例。我喜欢理解解决方案背后的原理。
```

### 4. 设定目标
```
我的目标是成为高级开发工程师，并提升系统架构设计能力。
```

## 档案结构

### 基本信息
- 职位角色和经验年限
- 主要技术和工具
- 专注领域和兴趣点
- 沟通偏好

### 技术档案
- 编程语言和框架
- 工具和平台
- 当前学习目标
- 技能水平评估

### 工作风格
- 问题解决方法
- 协作偏好
- 代码审查风格
- 学习方法

### 目标规划
- 短期目标
- 长期职业目标
- 当前项目
- 技能发展领域

## 最佳实践

### 档案创建
1. **具体明确**：提供具体的细节和示例
2. **实事求是**：准确反映技能水平
3. **长远考虑**：考虑未来目标和发展规划
4. **包含背景**：分享工作环境和约束条件

### 档案维护
1. **定期更新**：每月检查和更新
2. **添加示例**：记录成功的交互案例
3. **跟踪进度**：记录技能提升和成就
4. **优化偏好**：调整沟通风格偏好

### 隐私安全
- 所有档案数据保留在本地工作空间
- 不会向外部传输个人信息
- 您完全控制存储和分享的信息
- 档案可以加密或排除在备份之外

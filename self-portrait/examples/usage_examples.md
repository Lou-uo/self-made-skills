# Self-Portrait Usage Examples

## Example 1: Software Developer Profile

### Initial Setup
```
I'm a mid-level Python developer with 3 years of experience. I work primarily with Django and React, and I'm currently learning cloud technologies like AWS. I prefer detailed explanations with code examples, and I like to understand the "why" behind solutions.
```

```json
{
  "basic_info": {
    "role": "Mid-level Python Developer",
    "experience_years": 3,
    "primary_technologies": ["Python", "Django", "React"],

    "explanation_preference": "detailed_with_examples"

  "technical_profile": {
    "programming_languages": ["Python", "JavaScript"],
```

## Updating Profiles

### Adding New Skills
```
Update my profile: I've started learning TypeScript and have been working with Next.js for the past 6 months. I'm also getting into GraphQL.
```

### Changing Work Style
```
Update my work style: I now prefer more collaborative coding sessions and benefit from pair programming approaches. I've shifted towards test-driven development.
```

### Adding Goals
```
Add to my goals: I want to become a team lead within the next 2 years and improve my system design skills significantly.
```

## Profile Insights

The self-portrait skill can generate insights based on the profile:

### Learning Recommendations
Based on a developer's focus areas and current skills, the skill can recommend:
- Specific courses or resources
- Practice projects
- Skill progression paths

### Communication Optimization
Based on interaction patterns, the skill can suggest:
- Optimal explanation formats
- Preferred code review approaches
- Effective collaboration methods

### Career Development
Based on goals and current profile, the skill can provide:
- Skill gap analysis
- Career path suggestions
- Learning roadmap recommendations

These examples demonstrate how the self-portrait skill adapts to different user types and continuously evolves based on user interactions and stated preferences.

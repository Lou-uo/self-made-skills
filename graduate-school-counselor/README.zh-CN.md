# 研究生申请咨询师技能 (Graduate School Counselor)

一个专为本科生提供保研咨询和申请指导的综合性AI技能。

## 🎯 技能概述

研究生申请咨询师技能帮助学生：
- **精准定位**：基于个人背景推荐合适的院校档次
- **信息整合**：实时搜索最新保研信息和申请动态
- **策略制定**：制定个性化的申请策略和时间规划
- **竞争力分析**：全面评估申请竞争力并提供提升建议
- **研究组匹配**：推荐匹配的研究方向和导师信息

## 🌟 核心功能

### 1️⃣ **个性化档案管理** 📋
- **学术背景记录**：GPA、排名、专业课程
- **研究经历管理**：论文、项目、实验室经历
- **获奖成就追踪**：竞赛、奖学金、证书
- **语言成绩管理**：TOEFL、IELTS、GRE、CET等
- **目标学校跟踪**：关注院校动态和申请进度

### 2️⃣ **智能院校推荐** 🎓
- **分层推荐系统**：
  - **冲刺院校**：挑战更高目标
  - **匹配院校**：成功概率较高
  - **保底院校**：确保录取机会
- **多维度匹配**：研究方向、录取要求、个人背景
- **录取概率分析**：基于历史数据和当前背景

### 3️⃣ **实时信息搜索** 🔍
- **院校排名信息**：CS、EE等专业的权威排名
- **夏令营通知**：最新夏令营报名信息和截止日期
- **申请经验分享**：成功申请者的经验和建议
- **项目详细信息**：课程设置、导师信息、就业数据

### 4️⃣ **竞争力分析** 📊
- **综合评分系统**：GPA、研究经历、竞赛获奖等
- **优势劣势分析**：明确个人申请优势和不足
- **提升建议**：针对薄弱环节的具体改进方案
- **同类对比**：与成功申请者的背景对比

### 5️⃣ **申请策略制定** 📅
- **时间线规划**：从准备到入学的完整时间规划
- **优先级行动**：关键时间节点的重点行动
- **材料准备清单**：简历、个人陈述、推荐信等
- **面试准备指导**：常见问题、回答策略、技巧

## 🚀 使用场景

### 📝 **创建申请档案**
```
"我想创建我的研究生申请档案，我的背景是..."
```
技能将引导您逐步完善个人信息。

### 🎯 **获取院校推荐**
```
"基于我的背景，我应该申请哪些学校？"
```
技能将根据您的竞争力提供分层院校推荐。

### 🔍 **搜索最新信息**
```
"请帮我查找清华大学计算机系的夏令营信息"
```
技能将搜索并提供最新的夏令营和申请信息。

### 📊 **竞争力评估**
```
"请分析一下我的申请竞争力如何？"
```
技能将提供详细的竞争力分析报告。

### 📅 **制定申请策略**
```
"请为我制定一份申请策略和时间规划"
```
技能将生成个性化的申请策略和时间线。
